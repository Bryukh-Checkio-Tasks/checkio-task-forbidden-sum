我々の新しい計算機は検閲下に置かれており、検閲に係る特定の単語を受け付けない様になっている。 しかし、諸君にはこの検閲をなんとか誤魔化し、数字の合計を計算するプログラムを書いて貰いたいのだ。

諸君には数字のリストが与えられるので、それら数字の合計を計算するプログラムを書いてくれ。 但し、このプログラムでは、検閲される禁止ワードを含むことが出来ない。それだけでなく、この禁止ワードを含む単語も禁止だ。

検閲される禁止ワードのリストは次の通り:

  • sum
  • import
  • for
  • while
  • reduce

Input: 数字のリスト

Output: 与えられた数字の合計

Example: 

checkio([1, 2, 3]) == 6
checkio([2, 2, 2, 2, 2, 2]) == 12

How it is used: この課題は、あなたの創造力への挑戦です。 このような制限下にある仕様であってもクリアする解決法を思いつけるかどうか挑戦して下さい。

Precondition: The small amount of data. Let's creativity win!
